# 快速開始：自動粘貼功能

## 🚀 5 分鐘快速設置

### 步驟 1: 安裝依賴（1 分鐘）

```bash
cd macos_menubar_app
pip install -r requirements.txt
```

### 步驟 2: 設置 API Key（30 秒）

```bash
export OPENAI_API_KEY='your-openai-api-key'
```

### 步驟 3: 測試功能（1 分鐘）

```bash
python3 test_auto_paste.py
```

這個測試會：
- ✅ 檢查輔助功能權限
- ✅ 檢測焦點應用
- ✅ 測試剪貼板
- ✅ 測試自動粘貼

### 步驟 4: 授予權限（2 分鐘）

如果測試提示需要權限：

1. 打開「系統偏好設置」→「安全性與隱私」→「隱私」
2. 選擇左側的「輔助功能」
3. 點擊左下角的鎖圖標解鎖
4. 添加 `Python` 或 `Terminal`
5. 確保勾選框已選中

### 步驟 5: 運行應用（30 秒）

```bash
python3 speech_to_clipboard.py
```

## 🎯 快速測試自動粘貼

### 測試 1: 在 Notes 中使用

1. **打開 Notes** 應用
2. **創建新筆記**
3. **點擊編輯區域**（確保光標在輸入框中）
4. **按 ⌘R** 開始錄音
5. **說話**：「這是一個自動粘貼測試」
6. **按 ⌘R** 停止錄音
7. **查看結果**：文字應該自動出現在 Notes 中！

### 測試 2: 在 Slack 中使用

1. **打開 Slack**
2. **選擇任意頻道**
3. **點擊輸入框**
4. **按 ⌘R**，說：「這是來自語音輸入的消息」
5. **停止錄音**
6. **查看**：消息自動填入輸入框

## ⚙️ 常用設置

### 開啟/關閉自動粘貼

1. 點擊狀態列的 🎤 圖標
2. 選擇「設定」
3. 點擊「自動粘貼到焦點應用」切換

- **✓ 自動粘貼到焦點應用** - 已開啟
- **自動粘貼到焦點應用** - 已關閉

### 更改語言

1. 點擊狀態列圖標
2. 選擇「設定」→「語言: 自動偵測」
3. 輸入語言代碼（zh、en、ja）或留空自動偵測

## 🎤 使用技巧

### 最佳實踐

1. **錄音前確認光標位置**
   - 點擊目標輸入框
   - 確認光標在正確位置

2. **使用快捷鍵提高效率**
   - `⌘R` - 開始/停止錄音
   - 全程無需鼠標操作

3. **短句更準確**
   - 推薦：5-30 秒的錄音
   - 避免：超過 1 分鐘的長錄音

4. **檢查通知確認**
   - 每次粘貼後查看通知
   - 確認粘貼到了正確的應用

## ❓ 常見問題快速解答

### Q: 沒有自動粘貼？

**檢查清單：**
- [ ] 輔助功能權限已授予？
- [ ] 自動粘貼功能已開啟（設定中有 ✓）？
- [ ] 光標在輸入框中？
- [ ] 目標應用是前台應用？

### Q: 權限對話框沒有出現？

```bash
# 手動觸發權限請求
python3 test_auto_paste.py
```

### Q: 想要只複製不粘貼？

1. 點擊狀態列圖標
2. 設定 → 取消勾選「自動粘貼到焦點應用」
